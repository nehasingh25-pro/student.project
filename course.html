Course Details

Web Designing Fundamentals

Course Description

This course introduces students to the basics of web design. It covers HTML, CSS, and responsive layout techniques used to create professional web pages.

Course Information

Instructor: Neha Singh

Duration: 6 Months

Level: Beginner

Topics Covered

  • Introduction to HTML
  • CSS Styling and Layouts
  • Responsive Web Design
  • Mini Project: Personal Portfolio

Learning Outcomes

  • Understand website structure using HTML
  • Style web pages using CSS
  • Build responsive and user-friendly websites
